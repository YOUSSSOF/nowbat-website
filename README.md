# Nowbat Website

Official website for the **Nowbat (نوبت)** WordPress booking plugin — documentation, marketing, and live demo.

Built with Next.js 15, Tailwind CSS, and next-intl. Persian-first, RTL by default.

---

## Stack

- **Next.js 15** (App Router, SSG)
- **Tailwind CSS v3** with custom design tokens
- **next-intl** — Persian (`fa`) + English (`en`) i18n
- **MDX** — documentation and guides content
- **Framer Motion** — page animations

---

## Development

```bash
npm install
npm run dev       # http://localhost:3000
npm run build
npm run lint
```
